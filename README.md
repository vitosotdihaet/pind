# pind

## Расчет параметров эффективностей математической модели
Все реализовано на языке программирования Python, для установки, нужно выполнить следующие шаги:

```bash
python3 -m venv venv
source venv/bin/activate.sh
pip install -r requirements.txt
```

Далее можно запустить расчеты при помощи Jupyter Notebook в `math-model-calculations/main.ipynb`

Различные конфигурации класетра доступны в папке `math-model-calculations/configs`

## Запуск плагина

```bash
just pind
```
